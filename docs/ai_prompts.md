#  AI Prompts Used — MythCore RPG Launcher

Author: **Irvin Ociel Hernández Trejo**

This document contains example prompts used to generate and improve the database system using Artificial Intelligence tools.

---

#  Prompt List

##  Prompt 1 — Database Design

**Prompt:**

Create a relational database schema for an RPG launcher system that includes users, games, favorites, messages, friends, and reports.

**Purpose:**

To generate the initial database structure.

---

##  Prompt 2 — Normalization

**Prompt:**

Normalize this database structure to Third Normal Form (3NF) and explain each step.

**Purpose:**

To ensure the database followed normalization rules.

---

##  Prompt 3 — SQL Schema Creation

**Prompt:**

Generate SQL CREATE TABLE statements with primary keys and foreign keys for a gaming database system.

**Purpose:**

To build the main database structure.

---

##  Prompt 4 — Sample Data

**Prompt:**

Insert realistic sample data into users, games, and categories tables for testing purposes.

**Purpose:**

To populate the database with test records.

---

##  Prompt 5 — SQL Queries

**Prompt:**

Write SQL queries that use JOIN, COUNT, and GROUP BY to generate reports from a gaming database.

**Purpose:**

To create report queries.

---

##  Prompt 6 — ER Diagram

**Prompt:**

Generate a Mermaid ER diagram based on database tables and relationships.

**Purpose:**

To visualize database relationships.

---

##  Prompt 7 — Bug Testing

**Prompt:**

Create test cases to validate foreign keys and relational integrity in a MySQL database.

**Purpose:**

To simulate real database testing.

---

##  Prompt 8 — Documentation

**Prompt:**

Generate professional documentation files including README, dictionary, and normalization report.

**Purpose:**

To complete project documentation.

---

#  Summary

Artificial Intelligence tools were used to:

* Design database structure
* Generate SQL code
* Create documentation
* Validate database logic
* Improve system reliability

AI tools supported development but all final integration and validation were performed manually.
