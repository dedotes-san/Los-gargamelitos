# RPG Launcher Database Project

## Team

* Gargamel 

## Project Description

This project designs a relational database for an RPG Game Launcher platform.

The system allows users to:

* Manage their game library
* Store information about developers
* Categorize games by genres
* Track purchases and game ownership

The database follows relational design principles and is normalized to Third Normal Form (3NF).
